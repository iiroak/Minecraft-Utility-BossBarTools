# Boss Bar Tools

**A powerful and configurable Fabric mod that gives you complete control over boss bars in Minecraft 1.21.4.**

## 🎯 What does it do?

Tired of boss bars cluttering your screen? Boss Bar Tools lets you hide them completely or selectively filter which ones to show. Perfect for streamers, builders, or anyone who wants a cleaner HUD experience.

## ✨ Key Features

### 🔧 **Complete Control**
- **Global Toggle**: Hide all boss bars with a single keypress (default: `H`)
- **Smart Filtering**: Choose exactly which boss bars to hide based on their text
- **Instant Toggle**: Quick on/off switching with customizable keybind

### 🎯 **Advanced Filtering**
- **Exact Match**: Hide boss bars with specific titles like "Ender Dragon"
- **Contains Filter**: Hide any boss bar containing words like "loading" or "fish"
- **Regex Support**: Advanced pattern matching for power users

### ⚙️ **Easy Configuration**
- **ModMenu Integration**: Beautiful in-game configuration interface
- **Four organized tabs**: General, Exact Match, Contains, and Regex filters
- **Live Preview**: See changes immediately without restarting

### 🌍 **Quality of Life**
- **Multilingual**: Supports English and Spanish
- **Persistent Settings**: Your preferences are automatically saved
- **Performance Friendly**: Minimal impact using efficient Mixin injection

## 📖 How to Use

### Quick Start
1. Install the mod
2. Press `H` in-game to toggle all boss bars
3. Use ModMenu to access detailed configuration

### Advanced Filtering
Open the configuration menu and set up filters:

**Hide specific bosses:**
- Add "Ender Dragon" to Exact Match filters
- Add "Wither" to Exact Match filters

**Hide loading bars:**
- Add "loading" to Contains filters
- Add "progress" to Contains filters

**Hide percentage displays:**
- Add `\d+%` to Regex filters (matches "Loading 50%", etc.)

## 🔧 Technical Details

- **Client-side only**: Works on any server
- **Mod compatibility**: Works with all modded boss bars

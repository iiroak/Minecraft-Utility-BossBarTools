# Minecraft-Utility-BossBarTools

Mod de **Fabric** para Minecraft que permite ocultar boss bars, filtrarlas por texto y alternar su visibilidad mediante un keybind.

## Qué contiene

- **build.gradle**: configuración principal del proyecto.
- **gradle.properties** y `settings.gradle`: ajustes de compilación.
- **gradlew** y `gradlew.bat`: wrappers de Gradle.
- **README.md**: documentación principal.
- **MODRINTH_DESCRIPTION.md**: descripción pensada para publicación o distribución.
- **LICENSE**: licencia del proyecto.

## Funcionalidad general

Según el README y la estructura visible, el mod permite:

- ocultar boss bars por completo,
- aplicar filtros por coincidencia,
- alternar visibilidad con una tecla,
- mantener configuración persistente.

## Cómo compilar

```bash
./gradlew build
```

El `.jar` compilado debería quedar en `build/libs`.
